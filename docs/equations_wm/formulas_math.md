Linear Search: Finds the position of a target value within a list.
Binary Search: Efficiently finds the position of a target value within a sorted list.
Bubble Sort: Simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.
Selection Sort: Sorts an array by repeatedly finding the minimum element from the unsorted part and putting it at the beginning.
Insertion Sort: Builds the final sorted array one item at a time by repeatedly taking the next element and inserting it into the correct position.
Merge Sort: Divides the unsorted list into n sublists, repeatedly merges sublists to produce new sorted sublists until there is only one sublist remaining.
Quick Sort: Divides the array into smaller partitions, then recursively sorts each partition.
Heap Sort: Builds a binary heap and uses it to sort the elements in the array.
Radix Sort: Sorts numbers by processing individual digits from the least significant to the most significant.
Depth-First Search (DFS): Explores as far as possible along each branch before backtracking.
Breadth-First Search (BFS): Visits all the vertices of a graph in breadth-first order.
Dijkstra’s Algorithm: Finds the shortest path between two nodes in a weighted graph.
A* Algorithm: A heuristic search algorithm used for finding the shortest path between two points.
Bellman-Ford Algorithm: Finds the shortest paths from a single source node to all other nodes in a weighted graph.
Floyd-Warshall Algorithm: Finds the shortest paths between all pairs of vertices in a weighted graph.
Kruskal’s Algorithm: Finds a minimum spanning tree in a connected, undirected graph.
Prim’s Algorithm: Constructs a minimum spanning tree for a weighted, connected, undirected graph.
Knapsack Problem (0/1): Given weights and values, determines the maximum value that can be accommodated in a knapsack of fixed capacity.
Greedy Algorithm: Builds up a solution piece by piece, always choosing the next piece that offers the most immediate benefit.
Dynamic Programming: Solves problems by breaking them down into simpler subproblems and solving each subproblem only once.
Hashing: Maps data of arbitrary size to fixed-size values, typically for fast data retrieval.
Trie: Tree-like data structure used to store a dynamic set or associative array.
Bloom Filter: Probabilistic data structure used to test whether an element is a member of a set.
B-tree: Self-balancing tree data structure that maintains sorted data and allows searches, insertions, deletions in logarithmic time.
R-tree: Tree data structure used for spatial access methods.
PageRank Algorithm: Algorithm used by Google Search to rank web pages in their search engine results.
RSA Algorithm: Asymmetric encryption algorithm widely used for secure data transmission.
K-Means Clustering: Divides a set of points into k non-overlapping subgroups (clusters).
Hierarchical Clustering: Builds a hierarchy of clusters by either recursively merging or splitting existing clusters.
Naive Bayes Classifier: Probabilistic algorithm based on Bayes’ theorem for classification.
K-Nearest Neighbors (KNN): Classifies objects based on the majority class of their k nearest neighbors.
Support Vector Machine (SVM): Classifies data by finding the hyperplane that best separates classes.
Decision Tree: Hierarchical model of decisions and their consequences, used for classification and regression.
Random Forest: Ensemble learning method that constructs a multitude of decision trees and merges them together.
Gradient Boosting: Ensemble learning method that builds a model in a stage-wise fashion, optimizing for errors in the previous stage.
Principal Component Analysis (PCA): Technique for reducing the dimensionality of data while retaining most of its variability.
Apriori Algorithm: Finds frequent itemsets in a dataset for association rule mining.
Kruskal-Wallis Test: Non-parametric test for comparing three or more independent samples.
Expectation-Maximization (EM) Algorithm: Iterative method for finding maximum likelihood estimates of parameters in statistical models.
Monte Carlo Method: Uses random sampling to obtain numerical results, especially for complex problems.
Simulated Annealing: Optimization algorithm inspired by the annealing process in metallurgy.
Genetic Algorithm: Evolutionary algorithm inspired by natural selection that evolves solutions to optimization and search problems.
Ant Colony Optimization (ACO): Swarm intelligence technique for finding optimal paths.
Particle Swarm Optimization (PSO): Population-based optimization algorithm inspired by social behavior of birds and fish.
Backpropagation: Training algorithm for artificial neural networks.
Kohonen Self-Organizing Map: Unsupervised learning algorithm for dimensionality reduction and clustering.
Hopfield Network: Recurrent artificial neural network used for content-addressable memory.
Fisher’s Linear Discriminant Analysis: Linear method used for dimensionality reduction and classification.
Gaussian Mixture Model (GMM): Probabilistic model representing a mixture of Gaussian distributions.
Fast Fourier Transform (FFT): Efficient algorithm for computing the discrete Fourier transform.
Huffman Coding: Lossless data compression algorithm that uses variable-length codes.
Lempel-Ziv-Welch (LZW) Compression: Dictionary-based compression algorithm.
RSA Digital Signature Algorithm: Digital signature algorithm using asymmetric cryptography.
Elliptic Curve Cryptography (ECC): Public key cryptography based on the mathematics of elliptic curves.
Cholesky Decomposition: Decomposes a Hermitian, positive-definite matrix into the product of a lower triangular matrix and its conjugate transpose.
Levenshtein Distance: Measures the minimum number of single-character edits required to change one word into another.
Fast Marching Method: Numerical technique for solving the Eikonal equation.
Expectimax Algorithm: Used in game theory for decision making in partially observable environments.
Lloyd’s Algorithm (k-Means Clustering): Iterative algorithm to partition a dataset into k clusters.
Canny Edge Detector: Edge detection algorithm using multi-stage algorithm to detect a wide range of edges.
Mean Shift Algorithm: Non-parametric clustering algorithm that assigns each data point to the cluster whose mean is closest to it.
Gibbs Sampling: Markov Chain Monte Carlo (MCMC) algorithm for obtaining a sequence of observations that are approximated from a specified multivariate probability distribution.
Bellman-Ford-Moore Algorithm: Generalization of the Bellman-Ford algorithm that handles negative weight cycles.
Fisher-Yates Shuffle: Algorithm for generating a random permutation of a finite sequence.
Expectation-Maximization for Gaussian Mixture Models (EM-GMM): Applies the EM algorithm to estimate the parameters of a Gaussian mixture model.
Conjugate Gradient Method: Iterative method for solving systems of linear equations.
Dempster-Shafer Theory: Mathematical theory for reasoning with uncertainty.
Gale-Shapley Algorithm (Stable Marriage Problem): Finds a stable matching between two equally sized sets of elements.
Ford-Fulkerson Algorithm (Max Flow): Computes the maximum flow in a flow network.
Cocktail Shaker Sort (Bidirectional Bubble Sort): Variation of the bubble sort that sorts the list in both directions.
Ford-Bellman-Moore Algorithm: Generalization of the Ford-Fulkerson algorithm for handling negative weight cycles.
Minimum Spanning Tree Clustering (MSTC): Clustering algorithm based on the minimum spanning tree of a graph.
Page Replacement Algorithms (e.g., LRU, FIFO): Used in operating systems to manage the contents of a page table.
Depth-First Branch and Bound: Solves combinatorial optimization problems.
Hierarchical Temporal Memory (HTM): Machine learning model based on the structure and function of the neocortex.
Viterbi Algorithm: Dynamic programming algorithm used to find the most likely sequence of hidden states.
Simplex Algorithm: Algorithm for linear programming problems.
Batch Gradient Descent: Optimization algorithm used in machine learning for finding the local minimum of a function.
Forward-Backward Algorithm (Baum-Welch): Applies the forward and backward algorithms to estimate the parameters of a hidden Markov model.
Ramer-Douglas-Peucker Algorithm: Simplifies a curve represented by a sequence of points.
Branch and Bound Algorithm: Solves optimization problems by iteratively dividing the solution space into subsets.
Nelder-Mead Algorithm: Iterative optimization algorithm for nonlinear optimization problems.
Metropolis-Hastings Algorithm: Markov Chain Monte Carlo (MCMC) algorithm for generating a sequence of samples from a probability distribution.
Smith-Waterman Algorithm: Local sequence alignment algorithm used in bioinformatics.
Kolmogorov-Smirnov Test: Non-parametric test for comparing two distributions.
Expectation Propagation: Approximate inference algorithm in probabilistic graphical models.
Vapnik-Chervonenkis Dimension: Measure of the capacity of a hypothesis class.
Beam Search: Heuristic search algorithm that explores a graph by expanding the most promising node in a limited set.
Cooley-Tukey FFT Algorithm: Efficient algorithm for computing the discrete Fourier transform.
Knuth-Morris-Pratt Algorithm: String-searching algorithm that searches for occurrences of a “word” within a main “text” string.
A* Search Algorithm: Finds a path from start node to goal node with the smallest cost.
Gauss-Seidel Method: Iterative method used to solve a system of linear equations.
Iterative Closest Point (ICP): Algorithm employed to minimize the difference between two clouds of points.
Lagrange Interpolation: Polynomial interpolation method to approximate a function.
Bellman-Ford-SPFA Algorithm: Shortest path algorithm similar to Bellman-Ford but with optimizations.
Skip List: Data structure that allows for quick search, insertion, and deletion of elements.
Fisher’s Exact Test: Statistical test used to determine if there are nonrandom associations between two categorical variables.
Burrows-Wheeler Transform (BWT): Text transformation that precedes data compression algorithms like the Burrows-Wheeler Compression Algorithm.
Batch Normalization: Technique used to improve the training of deep neural networks.
Hough Transform: Technique used in image processing to detect shapes, particularly lines and circles.
